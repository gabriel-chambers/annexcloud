With EP PushCrew, You can add PushCrew (now VWO Engage) browser push notifications to your website in less than a minute.

== Description ==
Installing the EP PushCrew plugin will automatically insert the PushCrew Smart Code on your WordPress website. To get started, you just have to copy the PushCrew Account Id(Your Account ID can be found in the Account Details section under Settings area in the PushCrew dashboard) and paste it in this plugins’s settings. You can also exclude any page or post from showing post nutification in browser from metabox. You can enable that metabox to any custom post type too. By default it is active for post and page.

What is PushCrew?

PushCrew is a web-based push notification platform used by SaaS and B2B marketers, online content publishers, and eCommerce store owners. ... The platform aims to help them get higher clickthrough rates and expand their email marketing list. With PushCrew, users will be able to communicate better with their customers.

= Features: =
✔️ Add PushCrew script
✔️ Exclude script from Specific Pages/Posts
✔️ Custom Post Type Supported
✔️ Super Lightweight
 
== Installation ==

To install EP PushCrew, follow these steps:

1. Download and unzip the plugin

2. Upload ep-pushcrew to the /wp-content/plugins/ directory.

3. Activate the plugin through the 'Plugins' menu in WordPress
and you all set up.
4. Goto Setting>Ep PushCrew and set your pushcrew account ID. Now sit back and enjoy!!
